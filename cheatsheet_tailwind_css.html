üöÄ Quick Start

Tailwind CSS is a utility-first CSS framework. Classes are composed of:

  • Property: What you're styling (text, bg, p, m, etc.)
  • Value: The value to apply (red-500, 4, full, etc.)
  • Modifiers: Responsive (sm:, md:, lg:) or state (hover:, focus:)
<!-- Example: Responsive padding with hover effect --> <div class="p-4 md:p-8 hover:bg-gray-100"> Content </div>
